sopro-pager
===========

This component is a content area for multiple views or pages of content.

Usage
-----

    $ git clone https://github.com/sopro-dist/sopro-pager.git
    $ cd sopro-pager/src
    $ npm install
    $ grunt
    $ firefox ../component/test.html

Modes of display:
  Linear
  Accordion

TOC's/index:
  Tabs at top
  Progress >> _Arrows_ >> Top
  None
